# Changelog

Stay up to date with the latest changes to buildwhatever.

<Update label="Public Beta Launch" date="2026-02-24" tags={["launch", "platform"]}>
  buildwhatever is now in public beta! Sign up at [buildwhatever.dev](https://buildwhatever.dev) to get early access.

  **What's included:**
  - AI-powered workspace creation from natural language descriptions
  - Track and milestone system for structured project execution
  - 11 product development nodes (guides and tools)
  - Integrations with GitHub, Slack, Sentry, and email
  - Browser extension and VS Code extension
  - CLI and MCP server for AI-native workflows
</Update>

<Update label="Integration Ecosystem" date="2026-02-20" tags={["integrations"]}>
  Launched webhook-based integrations to connect your existing tools:

  - **GitHub**: Auto-create tasks from issues with priority mapping
  - **Slack**: Capture feedback via trigger words and emoji reactions
  - **Sentry**: Turn errors into tasks with deduplication
  - **Email**: Route support emails to your task board
</Update>
