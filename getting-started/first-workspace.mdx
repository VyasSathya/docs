# Create Your First Workspace

A **workspace** is your project container. Each workspace comes with an AI-generated plan, tracks, milestones, tasks, and recommended tools.

## What's in a Workspace?

- **Profile** â€” a guided workflow tailored to your goal (e.g., "Launch Coaching Business", "Build SaaS App")
- **Tracks** â€” parallel workstreams (e.g., Discovery, Marketing, Build)
- **Milestones** â€” key steps within each track
- **Nodes** â€” mini-apps and guides to help you succeed
- **Tasks** â€” your to-do list and progress tracker

## Creating a Workspace

### Step 1: Describe Your Goal

After signing in, you'll see: **"What do you want to do?"**

Be specific â€” the AI uses this to recommend the right profile and tools.

**Examples:**
- "Start a health coaching business"
- "Sell my art online"
- "Build a SaaS app for project management"
- "Launch a newsletter about comedy writing"

### Step 2: Review Your Generated Profile

The AI analyzes your goal and creates a custom workflow. You'll see:

- **Profile name** â€” e.g., "Launch Coaching Business"
- **Tracks** â€” parallel workstreams with milestones
- **Reasoning** â€” why this profile was chosen

Edit the workspace name if you want something different.

### Step 3: Choose Workspace Type

**Local workspace:**
- Connect to a folder on your computer
- The AI assistant can access files in that folder
- Adds terminal and file explorer tabs
- Best for: coding projects, file-heavy work

**Cloud workspace:**
- No folder connection needed
- Hides terminal and file explorer tabs
- Best for: planning, research, service businesses

### Step 4: Create

Click **"Create Workspace"** and you're in.

## Managing Multiple Workspaces

Have multiple projects? Create multiple workspaces.

**Switch between them via:**
- The sidebar dropdown (top left)
- Keyboard shortcut: `Ctrl+K` then type the workspace name

Each workspace maintains separate tasks, progress, and AI context.
